# Node-Babel Starter Pack

> A Simple node.js/express server that uses babel and babel-watch. Babel allows you to use the ES Module syntax (import) on the backend.

## Quick Start

```bash
# Install dependencies
npm install

# Serve on localhost:5000
npm run server
```
